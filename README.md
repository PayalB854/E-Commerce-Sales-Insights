# 📊 E-Commerce Sales & Profitability Analysis
## 📌 Project Overview
Analyzed 5,000+ sales records to identify loss-making regions and high-profit product categories.
## 🛠️ Tools
- Excel (Pivot Tables, VLOOKUP)
- Python (Pandas)
## 💡 Key Insights
- Identified a 12% loss in the 'Furniture' category due to high shipping costs.
- Recommended stock increase for 'Technology' products in Western regions.
